Companies API - Endpoints

Companies

GET /api/companies

Returns an array of all companies

GET /api/companies/random

Returns a random company object

GET /api/companies/:id

Returns the company with the specified id

Financial Data

GET /api/financials

Returns an array of all financial reports

GET /api/financials/:id

Returns an array of all financial reports for the specified company

Industry Data

GET /api/industries

Returns an array of all industries

Employees

GET /api/employees

Returns an array of all employees

GET >/api/employees/:companyId

Returns an array of all employees of the specified company
